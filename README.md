## How to Install:

### 1. Download the Theme File
- Navigate to the Releases section on the right side of this repository.
- Click on the latest release.
- Download the `cs50-dark-0.0.1.vsix` file.

### 2. Install in VS Code
- Open Visual Studio Code.
- Open the Command Palette by pressing `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS).
- Type `Extensions: Install from VSIX` and press enter.
- Locate and select the downloaded `cs50-dark-0.0.1.vsix` file.

Now you can select the theme from the `Themes` menu.
